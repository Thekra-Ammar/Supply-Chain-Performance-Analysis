## 📌 Project Overview
This project analyzes the logistical and commercial performance of a multi-regional supply chain using Tableau Desktop. Interactive dashboards were developed to monitor key KPIs such as revenue, costs, delivery time, and cancellation rate.

## 🎯 Objectives
- Analyze supply chain performance
- Track key business and logistics KPIs
- Detect anomalies in performance
- Evaluate supplier reliability

## 📈 Key Insights
- Identification of a significant performance drop in November 2023
- Analysis of cost vs. delivery time consistency
- Evaluation of supplier stability across regions

## 💡 Recommendations
- Reduce lead time
- Optimize supplier relationships
- Improve operational efficiency

## 🛠️ Tools & Technologies
- Tableau Desktop
